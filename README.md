# Musician
Make a responsive-design for desktop , tablet and mobile
##  Project Overview

This project is a responsive landing page for a musician portfolio.  
The layout adapts to mobile, tablet, and desktop screen sizes using a mobile-first approach.

The goal of the project was to:

- Implement a Figma design accurately
- Apply responsive design principles
- Use CSS Grid and Flexbox
- Follow modern front-end best practices

##  Responsive Design Strategy

The project follows a **mobile-first approach**:

### Base Styles (Mobile – 320px+)
- Single column layout
- Centered text
- Navigation links hidden
- Hamburger icon visible
- Social icon hidden

### Tablet (768px+)
- Larger typography
- Navigation links visible
- Hamburger menu hidden
- Image scaled appropriately

### Desktop (1024px+)
- Two-column layout (Grid: 1fr 2fr)
- Text aligned left
- Social icon visible
- Content vertically balanced using Flexbox

Media queries are applied only when the layout changes according to the design.
